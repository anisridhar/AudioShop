AudioShop readme file

High Level Explanation of Application:

AudioShop allows the user, a flutist, to record themselves playing music and denoise the track by having a video track the player’s fingerings, and record note data. The user can also save the resulting track to a file along with the original recording. 

How to Run:

Run the file termProjGUI5.py from the directory it is in. 
Key modules: openCV - must install through Canopy’s Package Manager
	     pyAudio - in folder in directory
	     pyDub - in folder in directory

File is typically run from Terminal, so all packages are consolidated. 

For more information on code design and functionalities, see the design documents.
